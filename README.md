 * grep
 * gtk 3
 * libtimezonemap

 * mhwd
 * parted (e2fsprogs, dosfstools, jfsutils, ntfs-3g, ntfsprogs, reiserfsprogs, xfsprogs)
 * pam_encfs
 * python-pyparted
 * python-configobj
 * python 3
 * python-gobject 3
 * python-dbus
 * python-cairo
 * python-yaml
 * rsync
 * sudo
 * systemd
 * ttf-aboriginal-sans
 * ttf-indic-otf
 * ttf-khmer
 * ttf-lohit-fonts
 * ttf-myanmar3
 * ttf-thaana-fonts
 * ttf-tlwg
 * upower
 * util-linux
 * webkitgtk 3
